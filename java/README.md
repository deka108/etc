## Version Management
#### Choice
[sdkman](https://github.com/jenv/jenv)

#### Installation and Setup
```curl -s "https://get.sdkman.io" | bash
source "$HOME/.sdkman/bin/sdkman-init.sh"```

#### Installing Java
`sdk install java`

#### Installing Spark
`sdk install spark`

## Dependency Management (Java)

#### Installation
